# Contributing

Thank you for taking an interest in the NDMO Governance Intelligence Platform.

## Before opening a change

- Use synthetic data only. Never commit organizational evidence, credentials, personal data, or database exports.
- Keep the independent-project disclaimer intact; this repository is not an official NDMO product or certification service.
- Add both Arabic and English interface text when introducing user-facing content.
- Preserve evidence traceability and human review when changing assessment behavior.

## Development workflow

1. Create a focused branch from `main`.
2. Install the backend and frontend dependencies described in the README.
3. Make a small, documented change.
4. Run the backend tests and the frontend test, lint, and build commands.
5. Open a pull request explaining the user impact, test coverage, and any schema or security considerations.

Please open an issue before proposing major schema changes, authentication changes, or a new AI provider.

By contributing, you agree that your contribution may be distributed under this repository's license.
